## ?? Formato de Documentación de Fixes

Cada fix debe incluir:
1. **Descripción del problema**
2. **Síntomas** (cómo se manifiesta)
3. **Causa raíz**
4. **Solución** (paso a paso)
5. **Prevención** (cómo evitarlo en el futuro)

## ?? Plantilla para Nuevos Fixes

```markdown
### Paso 2

- [Documento relacionado](...)
```
